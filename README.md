## David's Crack Heads

A silly web game where you play as a job application hunting down crack heads who are collecting coins on the streets.

### How to Play
- Click the brain emoji to start
- Read the rules (shown for 10 seconds)
- Use arrow keys to move your job application
- Collide with crack heads to hunt them and score points
- Crack heads roam and collect coins

### Access the Game
The game is hosted on GitHub Pages: [https://clothing1.github.io/David-/](https://clothing1.github.io/David-/)

Make sure the repository is public to access the link.
